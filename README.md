AMP KART Tools Dashboard

This repository contains the core frontend system for AMP KART – Advanced Market Platform, designed to manage essential eCommerce tools in a clean and efficient dashboard.

📂 Project Structure

root/
│
├── Photo/
│   └── logo.png          # Brand logo used across the dashboard
│
├── index.html            # Main dashboard UI (control panel for all tools)
├── calculator.html       # Pricing Calculator (profit & cost calculation tool)
├── compressor.html       # Image Compressor (optimize product images)
├── qr.html               # QR Code Generator interface
├── qr.js                 # QR logic & functionality

🚀 Features

- Pricing Calculator → Helps calculate product pricing, profit margins, and delivery costs
- Image Compressor → Reduces image size without losing quality (faster website loading)
- QR Generator → Create high-quality QR codes for links, payments, and branding
- Clean Dashboard UI → Minimal, fast, and business-focused interface
- Modular Structure → Each tool works independently and can be scaled easily

🎯 Purpose

This system is built to:

- Simplify eCommerce operations
- Reduce manual calculations
- Improve speed & efficiency
- Create a scalable internal tool ecosystem

⚙️ Usage

1. Open "index.html" to access the dashboard
2. Click on any tool to use its functionality
3. Each tool runs independently via its respective HTML file
